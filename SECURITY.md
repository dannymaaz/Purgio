# Política de Seguridad

## Versiones Soportadas

Actualmente, solo se proporciona soporte de seguridad y parches para la última versión estable del producto:

| Versión | Soportada |
| ------- | --------- |
| v1.x.x  | Sí        |
| < v1.0  | No        |

## Reportar una Vulnerabilidad

Valoramos la seguridad y la confianza de nuestros usuarios por encima de todo. Si descubres una vulnerabilidad de seguridad en Purgio, por favor **NO** abras un issue público. En su lugar, reporta el hallazgo de manera privada siguiendo estos pasos:

1. Envía un correo electrónico detallado a **dannymaaz@example.com** (o al correo oficial de contacto).
2. Describe detalladamente la vulnerabilidad, los pasos exactos para reproducirla y el impacto estimado.
3. Si es posible, proporciona un ejemplo de código de prueba o captura de pantalla.

Recibirás una respuesta en un plazo máximo de 48 horas reconociendo el reporte y coordinando el proceso de solución y divulgación responsable.
